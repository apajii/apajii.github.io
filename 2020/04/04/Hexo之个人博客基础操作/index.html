个人博客搭建操作分享

发表于 2020-04-04 分类于 技术

1.前期下载

1）node：https://nodejs.org/en/
2）Git：官网下载巨慢，这里推荐 https://blog.csdn.net/weixin_44198965/article/details/99686507
3）notepad：百度，其他软件也可以并非一定要使用这个，我暂且使用notepad

2.查看是否安装成功

安装步骤傻瓜式操作，安装完成后新建文件夹，在你新建的文件夹里面，右键点击Git Bash Here，
输入查看node版本：
    node -v 
输入查看npm版本：
    npm -v

3.安装

安装cnpm：
    npm install -g cnpm --registry=http://registry.npm.taobao.org 
查看cnpm版本：
    cnpm -v 
安装hexo框架：
    cnpm install -g hexo-cli 
查看hexo版本：
    hexo -v 
查看当前路径命令：
    pwd
初始化博客：
    hexo init    
本地访问地址：localhost:4000

这里记住四部曲：
    hexo clean##清理---hexo g##生成---hexo s##启动博客---Hexo d##部署到远端
创建新的文章：
    hexo n "文章题目" 
进入编写：
    vim 文章题目.md
保存退出：（这里推荐两种办法）1）先按esc键，输入:wp
                              2）按住shift+z+z
返回上一层：cd ..

4.配置

1）给git配置身份，因为我出现ERROR Deployer not found: git的情况，所以整理了一下把这段放在前面。
（用户名和邮箱地址是本地git客户端的一个变量，不随git库而改变）分别输入：
git config –global user.name “你的名字”
git config –global user.email “你的邮箱”

安装插件：
    cnpm install --save hexo-deployer-git

2）查看
查看目录文件：
git config –list
查看你的名字：
git config user.name
查看你的邮箱：
git config user.email

3）生成公钥
输入以下并按三个回车：
ssh-keygen -t rsa -C “你的邮箱”

4）打开C盘.ssh文件，打开rsa.pub(这个是公钥，不带pub的是私钥），打开你新建的文件夹下面的站点配置文件_config.yml，找到deploy，输入代码：

 type: git  
  repo: https://github.com/apajii/aspajii.github.io.git
  branch: master

部署到远端：Hexo d

5.下载主题

下载yilia主题到本地（当然网上有好多主题，你可以下载别的，我的网站还在不断美化中）
    git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia 
修改hexo根目录下的 _config.yml 文件 ： theme: yilia
查看博客：https://你的用户名.github.io
